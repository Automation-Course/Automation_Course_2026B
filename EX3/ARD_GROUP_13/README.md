# README - קבוצה 13

## פרויקט: שומר המָן במדבר 🌾

מערכת בקרת משקל מבוססת Arduino לאיסוף מָן במדבר, המודעת ליום בשבוע.
ימים א'–ה': יעד 200g. יום ו': יעד 400g (משנה). שבת: המערכת נעולה.

---

## הוראות הפעלה

1. חברו את הארדואינו למחשב או למקור מתח מתאים.
2. וודאו שמשטח השקילה ריק בזמן ההפעלה.
3. המתינו להודעת הפתיחה במסך LCD: "Manna Guardian / Booting...".
4. אם צריך לקדם את היום בשבוע — לחצו על הכפתור המחובר לפין D7 (יום הבא).
5. אם צריך לעבור למשפחה הבאה — לחצו על הכפתור המחובר לפין D8 (סיום איסוף).
6. הצמידו תג RFID של המשפחה לקורא לזיהוי. בסביבת Tinkercad — לחצו על הכפתור המחובר ל-A1 לדמות סריקה.
7. הניחו את כלי האיסוף על משטח השקילה.
8. עקבו אחרי ההודעות במסך LCD:
   - **Place manna** — אין משקל משמעותי, התחל לאסוף.
   - **Gather more** — חסר משקל, המשך לאסוף.
   - **Perfect** — המשקל בטווח התקין; ה-LED הירוק נדלק וה-Servo פותח את האסם.
   - **Do NOT hoard** — נאספו יותר מדי; ה-LED האדום נדלק והבאזר מצפצף.
9. כאשר המשקל תקין, יישמע אישור ויש לוחצן ללחוץ על D8 (NEXT) כדי לעבור למשפחה הבאה.
10. אם רוצים לאפס את המשקל — לחצו על כפתור D8 (Tare מתבצע אוטומטית).
11. בשבת (Day = Sat) המערכת נעולה; ה-LCD מציג "** SHABBAT ** / No collection" וה-Servo נשאר סגור.
12. כל מעבר מצב נרשם ל-Serial Monitor (9600 baud) לתיעוד.

---

## קוד הפרויקט

קוד המקור המלא נמצא בקובץ:

**`manna_guardian.ino`**

הקוד תומך בשני מצבי הפעלה דרך הפלאג `SIMULATION_MODE`:

- **`#define SIMULATION_MODE 1`** — סביבת Tinkercad. חיישן המשקל מוחלף בפוטנציומטר על A0, וקורא ה-RFID מוחלף בכפתור על A1.
- **`#define SIMULATION_MODE 0`** — חומרה פיזית. משתמש ב-HX711 לחיישן המשקל וב-RC522 לזיהוי RFID.

## פינים וחיבורים

| רכיב | פין | תפקיד |
|---|---|---|
| LCD I2C 16×2 | A4 (SDA), A5 (SCL) | תצוגת מצב, יום, משקל ויעד |
| LED ירוק | D4 | חיווי "משקל בטווח התקין" |
| LED אדום | D5 | חיווי "אגירה" / שבת |
| Buzzer | D6 | אזעקה במצב OVER |
| Button "יום הבא" | D7 | קידום היום בשבוע |
| Button "סיום איסוף" | D8 | מעבר למשפחה הבאה |
| Servo SG90 | D9 (PWM) | פתיחת/סגירת דלת אסם |
| HX711 + Load Cell (פיזי) | D2 (DT), D3 (SCK) | מדידת משקל |
| Potentiometer (סימולציה) | A0 | מדמה משקל 0–500g |
| RFID RC522 (פיזי) | SS=D10, MOSI=D11, MISO=D12, SCK=D13, RST=A0, VCC=3.3V | זיהוי משפחה |
| Button "סרוק תג" (סימולציה) | A1 | מדמה סריקת RFID |

**הערה:** RC522 פיזי דורש מתח **3.3V בלבד**. חיבור ל-5V עלול לשרוף את הרכיב.

---

## בדיקת התקינות

1. הפעלו את הארדואינו והמתינו לאתחול.
2. בדקו שה-LCD מציג את היום הנכון. אם לא — לחצו D7 עד היום הנכון.
3. בצעו סריקת RFID (או לחיצה על A1 בסימולציה).
4. הניחו משקולת על משטח השקילה (או הזיזו את הפוטנציומטר בסימולציה).
5. עקבו אחר Serial Monitor — אמורות להופיע הודעות [SCAN], [STATE], [NEXT].
