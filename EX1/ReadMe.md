# 🧮 מחשבון המרת בסיסים (עשרוני ↔ הקסדצימלי)

מחשבון פייתון פשוט ויעיל לביצוע המרות בין **בסיס עשרוני (10)** ל**בסיס הקסדצימלי (16)** ולהיפך.

---

## ✨ תכונות עיקריות
* **המרה ידנית**: המימוש מתבצע באמצעות לוגיקה מתמטית עצמאית, ללא שימוש בפונקציות המרה מובנות של השפה.
* **דו-כיווני**: תמיכה מלאה בהמרה מעשרוני להקסדצימלי ומהקסדצימלי לעשרוני.
* **ממשק משתמש**: תפריט אינטראקטיבי להפעלה נוחה דרך הטרמינל.
* **חסינות**: טיפול במספרים חיוביים, שליליים ובדיקת תקינות קלט.

---

## 🚀 הוראות הרצה

### דרישות קדם
* התקנת פייתון (Python 3.x) על המחשב.

### הרצת המחשבון
1. שכפל את ה-Repository למחשב שלך (Clone).
2. פתח את הטרמינל או את ה-**Git Bash**.
3. הרץ את הפקודה הבאה:
   ```bash
   python main.py
