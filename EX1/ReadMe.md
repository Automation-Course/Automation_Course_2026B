# 🧮 Base Converter (Decimal ↔ Hex)

## מטרת התרגיל
[cite_start]במסגרת תרגיל זה, פותח מחשבון המרה בין בסיס עשרוני לבסיס הקסדצימלי (בסיס 16) ולהיפך[cite: 20]. [cite_start]מטרת העבודה היא להכיר את הלוגיקה המתמטית מאחורי בסיסי מספרים תוך תרגול עבודה עם כלי ה-Git וניהול גרסאות[cite: 2, 3].

## 🛠 תכונות המחשבון
* **המרת עשרוני להקסדצימלי:** המרה ידנית באמצעות חילוק חוזר בשארית.
* **המרת הקסדצימלי לעשרוני:** חישוב ערך פולינומי לפי חזקות של 16.
* [cite_start]**ללא פונקציות מובנות:** כל תהליך ההמרה נכתב מאפס ללא שימוש בפונקציות המרה מוכנות של Python (כמו `hex()` או `int(x, 16)`)[cite: 24].
* [cite_start]**טיפול בשגיאות:** המערכת מזהה קלט לא תקין ומציגה הודעת שגיאה מתאימה[cite: 21].

---

## [cite_start]🧠 הנחות יסוד בפיתוח [cite: 27]
1. המחשבון תומך במספרים שלמים (Integers) בלבד.
2. הקלט להמרה הקסדצימלית יכול לכלול אותיות קטנות או גדולות (A-F / a-f).
3. המחשבון תומך במספרים חיוביים ושליליים כאחד.
