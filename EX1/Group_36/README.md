\# Base Converter - Decimal & Hexadecimal (Group 36)

\## Description This is a Python-based interactive calculator that
converts numbers between Decimal (base 10) and Hexadecimal (base 16).
The conversion algorithms (Horner\'s method and repeated division) were
implemented manually without using Python\'s built-in base conversion
functions.

\## Features \* Convert Decimal to Hexadecimal. \* Convert Hexadecimal
to Decimal. \* Supports positive and negative numbers. \*
Case-insensitive hexadecimal input (supports both \'a-f\' and \'A-F\').
\* Includes a built-in automated test suite to verify edge cases.

\## How to Run 1. Ensure you have Python 3.x installed on your computer.
2. Open a terminal or command prompt. 3. Navigate to the folder
containing the script. 4. Run the script using the following command:
\`python base_converter.py\`

\## Usage Instructions Once the program is running, a menu will appear
on the screen: \* Choose \*\*1\*\* to enter a decimal number and get its
hexadecimal equivalent. \* Choose \*\*2\*\* to enter a hexadecimal
number and get its decimal equivalent. \* Choose \*\*3\*\* to run the
automated tests. \* Choose \*\*4\*\* to exit the program.
