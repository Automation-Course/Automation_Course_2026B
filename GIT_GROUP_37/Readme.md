**תיאור התוכנית**

תוכנית זו היא מחשבון הכתוב בשפת פייתון, המבצע המרה של מספרים בין בסיס עשרוני (רגיל) לבסיס הקסדצימלי (בסיס 16) בשני הכיוונים. ההמרות מבוצעות מאפס באמצעות אלגוריתמים מתמטיים בלבד, ללא שימוש בפונקציות המרה מובנות של השפה.



**הוראות שימוש במחשבון**

עם הפעלת התוכנית, היא תרוץ בשני שלבים ברצף:

שלב א': המרה מעשרוני להקסדצימלי

•	התוכנית תציג את ההודעה: Please enter a decimal number

•	המפעיל נדרש להזין מספר שלם וחיובי (לדוגמה: 255) וללחוץ Enter.

•	התוכנית תדפיס את התוצאה המומרת לבסיס הקסדצימלי.



שלב ב': המרה מהקסדצימלי לעשרוני

•	התוכנית תציג את ההודעה: Please enter a hexadecimal number

•	המפעיל נדרש להזין מספר הקסדצימלי תקין וחיובי (לדוגמה: FF או 1b) וללחוץ Enter. ניתן להזין אותיות גדולות או קטנות.

•	התוכנית תדפיס את התוצאה המומרת לבסיס עשרוני, ותסיים את ריצתה.



**טיפול בשגיאות קלט** 

התוכנית כוללת הגנות מפני קלט שגוי כדי למנוע קריסה:

•	קלט שאינו מספר שלם (כגון שבר או טקסט): התוכנית תציג הודעת שגיאה ותבקש מהמפעיל להזין קלט מחדש.

•	מספרים שליליים: התוכנית תזהה את סימן המינוס, תציג שגיאה הדורשת מספר חיובי, ותבקש קלט מחדש.

•	תווים שאינם חוקיים בבסיס 16 (כגון האות G): המערכת תתפוס את השגיאה, תתריע על פורמט לא חוקי, ותאפשר למפעיל לנסות שוב מבלי לצאת מהתוכנית.
